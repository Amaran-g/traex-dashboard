# TRAEX AI Admin Dashboard

🚀 Welcome to the official admin dashboard of **TRAEX AI** — Empowering Enterprises with Smart Data.

## 🌍 Live Site
[🔗 Visit the Admin Dashboard](https://admin.traexgglobal.in/)

## 📁 Project Structure
- `index.html`: Landing page  
- `/admin/dashboard.html`: Live dashboard interface  
- `/docs/`: Documentation (coming soon)  
- `/lk/`: Sinhala localized page (coming soon)

## 🛡️ Features
- Real-time analytics & user insights  
- Enterprise-grade security  
- Multilingual documentation support  
- Modern UI with Tailwind CSS + Alpine.js  
- GDPR compliant with Cookiebot & Consent Mode

## 🌐 Technologies Used
- TailwindCSS  
- Alpine.js  
- GitHub Pages (Hosting)  
- Google Analytics (Consent Mode Enabled)  
- Cookiebot CMP

## ⚙️ Setup & Deployment
Hosted with GitHub Pages  
➡️ Auto-deployment using GitHub Actions (`.github/workflows/static.yml`)
